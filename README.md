## ‚ú® Key Features

* **Immersive Editor**: A distraction-free, full-screen writing environment that automatically focuses on your content.
* **Intelligent Sorting**: Notes are prioritized by pinned status, ensuring your most important thoughts remain at the top.
* **Advanced Filtering**:
* **Dashboard**: View all active notes.
* **Favorites**: Access only your pinned/favorite items.
* **Archive**: Store notes you don't need immediately without deleting them.


* **Vibrant Categorization**: Automatically themed cards based on categories like Personal, Work, Study, and Ideas.
* **Persistence**: Full integration with `localStorage` to keep your notes and theme settings safe across sessions.
* **Smooth Motion**: Powered by Framer Motion for layout transitions and staggered list animations.

## üöÄ Installation & Setup

1. **Clone the repository**:
```bash
git clone https://github.com/YOUR_USERNAME/notes-app.git
cd notes-app

```


2. **Install dependencies**:
```bash
npm install

```


3. **Required Tailwind v4 Setup**:
Ensure your `vite.config.js` includes the Tailwind plugin and your CSS file imports Tailwind as shown in the project configuration.
4. **Start the development server**:
```bash
npm run dev

```
